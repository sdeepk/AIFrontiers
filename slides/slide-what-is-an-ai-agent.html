1.2 AI Agents

What Is an AI Agent?

A system that uses an LLM as its reasoning engine, has access to tools, takes multi-step actions, and operates with varying degrees of autonomy toward a stated goal.

Stress the precision here: an agent is not just a chatbot. It has tools (APIs, databases, code execution, browser), takes multi-step actions, and has autonomy. The key 2026 shift is from single agents to multi-agent systems — networks of specialised agents collaborating under an orchestrator. This distinction matters because multi-agent systems require fundamentally different architecture than a single LLM call.
