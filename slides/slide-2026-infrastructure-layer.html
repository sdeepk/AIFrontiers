The 2026 Infrastructure Layer

MCP

Model Context Protocol. Anthropic's open standard — now under Linux Foundation. The USB connector for agents: standard way to wire any model to any tool.

A2A

Agent-to-Agent Protocol. Google's standard for agent-to-agent communication. Allows specialised agents to hand off tasks and share context.
MCP is now hosted under the Linux Foundation's Agentic AI Foundation alongside OpenAI's AGENTS.md and Block's Goose framework — this signals it is the standard, not a proprietary play. If a client asks "how does our agent talk to Salesforce / Jira / SAP?" the answer is MCP. A2A handles agent-to-agent communication across organisational boundaries. The combination of MCP + A2A is the plumbing of the multi-agent enterprise. Investing in MCP expertise now gives you a 12-18 month advantage window.
