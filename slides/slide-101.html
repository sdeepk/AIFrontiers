The Minimum Viable Evaluation Stack

Golden dataset: 100–300 representative queries with expected outputs. Versioned in git.
LLM-as-judge pipeline: Claude Opus 4.6 or GPT-5.4 evaluating accuracy, groundedness, completeness.
Factual consistency score: Does every claim appear in source documents?
Regression gate: Every PR that changes a prompt must pass evaluation. Score drops >3% = PR blocked.
