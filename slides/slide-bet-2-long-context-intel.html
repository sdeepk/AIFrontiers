Bet 2: Long-Context Intelligence

Llama 4 Scout: 10M tokens. Retrieval pipeline can be replaced by putting entire knowledge base in context.
Invest in: Long-context evaluation, context architecture design, context caching
Impact: Happening right now. Redesign RAG pipelines with long context as a variable.
This is an architectural shift happening right now, not a future prediction. With 1M and 10M token context windows, the retrieval pipeline for many use cases can be replaced by simply putting the entire knowledge base in context. This changes the economics of document intelligence, contract analysis, and codebase understanding. The investment is in new evaluation strategies — 1M token contexts require fundamentally different testing than short-context systems. Also invest in context caching expertise, which reduces costs dramatically on repeated large-context calls.
