Fifteen minutes every Friday — this is not a long meeting. Each developer shares one thing AI helped them do better that week. The two questions — what worked, and what prompt or context change made it work — surface the team's collective prompt engineering intelligence. Document the best insights in the prompt library. This is how tacit AI knowledge becomes explicit team knowledge. The format is deliberately lightweight so it actually happens consistently. Start this next Friday.
