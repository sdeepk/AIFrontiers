The Code Review Prompt

Security vulnerabilities — injection, auth bypass, data exposure
Performance issues — N+1 queries, blocking calls, memory leaks
Error handling completeness — external calls, edge cases, null handling
Test coverage gaps — what scenarios are untested?
Context engineering opportunities — could structure make AI assistance better?

Flag by severity: CRITICAL / IMPORTANT / SUGGESTION

This is a ready-to-use prompt template for AI-assisted code review. The fifth item — context engineering opportunities — is unique: it asks whether comments, types, or structure could make AI assistance better in this part of the codebase. The severity flagging (CRITICAL, IMPORTANT, SUGGESTION) forces the AI to prioritize, which reduces noise. For CRITICAL and IMPORTANT items, it must state the specific risk and the minimal fix. Copy this into your team's prompt library and adapt it to your stack. Use it as a pre-merge gate alongside your evaluation pipeline.
