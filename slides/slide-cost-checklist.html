Cost Checklist

Token costs at scale consistently surprise everyone
Token usage instrumented per agent, per workflow step, per user
Model routing tiered: cheap model for routing, capable model for reasoning
Prompt caching enabled for repeated context
Batch API for non-real-time workloads (50% cost reduction)
Token costs at scale consistently surprise everyone — stress this. The missing item from the slide: model your monthly cost projection at 10x and 100x current usage before going to production. Anthropic prompt caching gives up to 75% cost reduction on repeated content like system prompts and reference documents. Both Anthropic and Google offer Batch APIs for non-real-time workloads at 50% cost reduction. The tiered routing pattern is critical: use a cheap fast model like Gemini Flash for classification and routing, then a capable model like Claude Opus for the actual reasoning.
