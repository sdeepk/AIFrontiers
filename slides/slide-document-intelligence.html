Agent Category
What's working: Contract review, regulatory parsing, financial extraction, RFP response generation
Evidence: Multi-modal agents processing complex structured documents
The unlock: 1M token context windows — large documents processed end-to-end without chunking
Caveat: Legal interpretation still requires human review
JP Morgan's COiN-era work has evolved — 2026 deployments use multi-modal agents (text + image + table) to process complex structured documents that earlier pipelines could not handle. The 2026 unlock is frontier models with 1M token context windows (Gemini 3.1 Pro, Claude Sonnet 4.6 beta) meaning large documents can be processed end-to-end without fragile chunking strategies. This changes document intelligence architecture significantly. Caveat: interpretation of legal language still requires human review — AI surfaces, humans decide.
