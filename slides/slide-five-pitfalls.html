The Five Pitfalls

Confident hallucination: Agent states facts with full confidence. Source says otherwise.
Prompt-injected agent: Retrieved doc says "Ignore instructions." No input sanitisation = vulnerable.
Cost runaway: Agentic loop fails to terminate. 50,000 LLM calls overnight. $40K invoice.
Context bleed: Agent retrieves User A's docs into User B's session.
Over-trusted agent: Human oversight removed after months of success. First major failure = disproportionate trust damage.
Walk through each with the solution. Confident hallucination: require citation and source matching as mandatory output schema. Prompt-injected agent: implement input sanitisation on all retrieved content before it enters context. Cost runaway: hard max-iteration limits and cost circuit breakers — the $40K overnight invoice is a real example. Context bleed: strict session isolation in all retrieval pipelines — this is especially critical in multi-tenant SaaS. Over-trusted agent: keep human checkpoints even after months of success — reduce the friction around them, but never remove the checkpoints themselves. Transition to 2.6: armed with these guardrails, let us talk strategy.
