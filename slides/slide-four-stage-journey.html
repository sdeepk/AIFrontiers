The Four-Stage Journey

1.6 Discovery → Scale
Stage 1 Discovery 2–4 weeks Problem definition. Clear use case agreed.
→
Stage 2 POC 4–8 weeks Technical feasibility. Working demo in sandbox.
→
Stage 3 Pilot 8–16 weeks Real users, real data. Measurable ROI.
→
Stage 4 Scale Ongoing Operationalise, monitor, evaluate, expand.
This is the complete customer journey model. Discovery is 2-4 weeks for problem definition. POC is 4-8 weeks for technical feasibility in a sandbox. Pilot is 8-16 weeks with real users, real data, and real feedback. Scale is ongoing — operationalise, monitor, evaluate, iterate, expand. Each stage has a clear success criterion. The most common failure pattern is jumping from POC directly to scale without a proper pilot phase. Let us walk through what makes each stage succeed.
