The Production Readiness Checklist

2.2 Scaling AI
Reliability — retry logic, circuit breakers, fallback routing
Evaluation — the most neglected category
Cost — token usage instrumented per agent, per workflow
Security — prompt injection protection on all retrieved content
Operations — full distributed tracing

Most teams only do the first one.

This checklist is updated specifically for agentic systems — traditional ML production checklists miss agent-specific concerns like runaway loops, tool call permissions, and agent state recovery. The next four slides break down each category in detail. Emphasise that most teams only address reliability and skip evaluation, cost, and security entirely. The evaluation gap is the single biggest reason AI systems degrade silently in production. Ask the audience: how many of these five categories does your current system cover?
