This is the recommended stack for 2026 — not exhaustive, but the starting point. Key choices: start with pgvector for your vector store since you likely already have PostgreSQL, then migrate to Weaviate as you scale. Ollama plus Llama 4 Scout handles sensitive data and offline development. LangGraph or LlamaIndex Workflows for stateful multi-agent pipelines. RAGAS plus Langfuse gives you both RAG evaluation and LLM tracing with cost monitoring. The MCP SDK is critical — building MCP integrations for client enterprise tools is where Covalense can create differentiated value. These tools work together as a coherent stack.
