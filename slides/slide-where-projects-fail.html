Walk through each briefly with the detail from the doc. "AI strategy without a use case" means "we need an AI roadmap" with no specific painful problem. Data readiness: 6-12 months of data prep is normal, not an exception. POC to production gap: the POC used GPT-5.4 with no latency/cost constraints and a clean test dataset — production has both constraints and messy real data. No evaluation: systems shipped without any way to know if they are working or degrading. No change management: the AI system is built but the workflow is unchanged and humans route around it. Metrics creep makes it impossible to declare victory. Prompt injection: retrieved documents can contain adversarial instructions — no sanitisation means critical vulnerability.
