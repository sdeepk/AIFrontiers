Efficiency Habits Habit 5

Model-Appropriate Task Routing

Don't drive a Formula 1 car to the grocery store.

Task Type Model Class
Route/classify a query Gemini Flash / Haiku ($0.10/M)
Simple code completion Copilot / Cursor Tab
Complex reasoning Claude Opus 4.6 / GPT-5.4 ($15+/M)
Long document analysis Gemini 3.1 Flash (1M context)
Multi-step agentic task Claude Opus 4.6 Agentic
Security review Claude Opus 4.6
Batch processing Batch API (50% cost reduction)
On-prem / sensitive data Llama 4 Scout via Ollama

Prompt caching: 90% cost reduction (Anthropic), 75% (Google Gemini) on repeated system prompts. Enable for architecture docs, coding conventions, and domain context.

The price difference is dramatic: a routing or classification task on Gemini Flash costs $0.10 per million tokens, while complex reasoning on Claude Opus 4.6 costs $15+ per million. Using Opus for everything is like driving a Formula 1 car to the grocery store. The prompt caching point is especially important: if you have a large system prompt with architecture docs and coding conventions that you include in every call, Anthropic's caching reduces cost by 90% on cached tokens and Google's by 75%. Enable this for any repeated context. Batch API gives 50% cost reduction for async workloads — use it for non-real-time jobs.
