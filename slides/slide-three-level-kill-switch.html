The Three-Level Kill Switch

Level 1: Feature Flag Toggle AI off without deployment
Level 2: Circuit Breaker Auto-disable on error rate threshold
Level 3: Emergency Stop Manual override with immediate effect
Every production AI system needs all three levels. Feature flag lets you toggle AI off without a deployment — this is for planned rollbacks or gradual rollouts. Circuit breaker auto-disables the system when the error rate crosses a threshold — this catches degradation before users report it. Emergency stop is manual override with immediate effect for when something goes seriously wrong. Additionally, implement model version pinning so you know exactly which model version, prompt version, and retrieval pipeline version produced any given output. For EU AI Act compliance, a defined human reviewer role with authority and access to intervene is mandatory for high-risk systems.
