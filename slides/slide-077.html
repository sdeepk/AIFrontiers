The Five Pitfalls

Confident hallucination: Agent states facts with full confidence. Source says otherwise.
Prompt-injected agent: Retrieved doc says "Ignore instructions." No input sanitisation = vulnerable.
Cost runaway: Agentic loop fails to terminate. 50,000 LLM calls overnight. $40K invoice.
Context bleed: Agent retrieves User A's docs into User B's session.
Over-trusted agent: Human oversight removed after months of success. First major failure = disproportionate trust damage.
