The Key Insight on Models

The four frontier models are within 3–5% of each other on most tasks. Model selection has become secondary to system design.

Emphasise this hard: a well-designed agent system on Gemini Flash outperforms a poorly designed agent system on GPT-5.4. The leverage is in the architecture — context engineering, retrieval quality, evaluation pipelines — not the model. This is counterintuitive for teams that spend weeks debating which model to use. Stop model shopping, start system designing. Claude Opus 4.6, GPT-5.4, Gemini 3.1 Pro, and Grok 4 all score within 3-5% of each other on most benchmarks.
